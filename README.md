# Student Report System ERD

## Description
This repository contains an Entity Relationship Diagram (ERD) for a Student Report System.

## GitHub Usage
The database structure was analyzed from a GitHub repository using migration files found in the database/migrations folder.

## Entities Identified
- Students
- Class Forms
- Subjects
- Exams

## Relationships
- One Class Form has many Students (1:M)
- One Subject has many Exams (1:M)

## Tools Used
- GitHub
- draw.io

## Files Included
- ERD Diagram
- Assignment Report
